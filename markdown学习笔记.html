列表

无序列表使用 * + -

*

  • 红
  • 黄
  • 蓝

+

  • 红
  • 黄
  • 蓝

-

  • 红
  • 黄
  • 蓝

有序列表使用数字-句号-空白， 序号不用连续，但会转换成连续的：

  1. 红
  2. 黄
  3. 蓝

文章中如果正好有数字-句号-空白，需要特殊转义：

1986. 需要添加一个\符号在数字和句号之间。

代码区块

const a = 1;
const b = 2;
console.log('a + b', a + b);

分割线

多于3个的 * 或 - 。

如果在行内的代码可以使用`来标记

使用print() 来表示打印操作。

链接

链接文字使用[方括号文字]后面加（圆括号链接）来标记

或者

链接文字使用[方括号文字]后面加[链接标号]来标记，

后面再使用[链接标记]： 链接 ‘提示文字’ 表示。

链接到百度

链接到 [example][id].

链接到 [example][id1].

链接到 [example][id2].
[id]: http://example.com/ “提示文字”
[id1]: http://example.com/ “提示文字1”
[id2]: http://example.com/ “提示文字2”

强调

使用 * 或 _ 表示强调，一个表示<em>斜体 二个表示<strong>加粗
